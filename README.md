# Olgusal

Text analysis of Turkish Prime Minister speeches before elections.

## Overview

This project analyzes political speeches and discourse patterns, focusing on pre-election statements by Turkish Prime Ministers.

## Purpose

- Analyze speech patterns and rhetoric
- Study political communication trends
- Document historical political discourse

## Technologies

- Java
- Natural Language Processing
- Text Analysis

## Data

Contains collected speeches and related text data for analysis.
